# How to Download
1- Locate downloader
![image]([https://github.com...](https://github.com/NePtuNee0/Quotes-Rainmeter/blob/7365e9206f78252e2c543d6051dd6115a30c44e0/image/1.png))

2- Copy Quotes folder 
![image]([https://github.com...](https://github.com/NePtuNee0/Quotes-Rainmeter/blob/484d3ff480c8ce1339920c74bdbdc0fafe7bda3e/image/2.png)

2- Go to Documents and locate rainmeter folder and  open it
![image]([https://github.com...](https://github.com/NePtuNee0/Quotes-Rainmeter/blob/484d3ff480c8ce1339920c74bdbdc0fafe7bda3e/image/3.png)

3- Paste Quotes folder inside the Skins folder 
![image]([https://github.com...](https://github.com/NePtuNee0/Quotes-Rainmeter/blob/484d3ff480c8ce1339920c74bdbdc0fafe7bda3e/image/4.png)

4- Open Rainmeter locate Quotes folder (press refresh all if can't see it) and press load
![image](https://github.com/NePtuNee0/Quotes-Rainmeter/blob/484d3ff480c8ce1339920c74bdbdc0fafe7bda3e/image/5.png)

5- Go to quotes_auto.py and right click open with python 
![image](https://github.com/NePtuNee0/Quotes-Rainmeter/blob/484d3ff480c8ce1339920c74bdbdc0fafe7bda3e/image/6.png)

6- If you want another Quote open quotes_auto.py again and refresh the skin
![image](https://github.com/NePtuNee0/Quotes-Rainmeter/blob/484d3ff480c8ce1339920c74bdbdc0fafe7bda3e/image/Result.png)
